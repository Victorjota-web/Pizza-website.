# 🍕 Pizza Order App

Craving some pizza? This app lets you add delicious options to your cart and simulate an order 😋  
No calories involved, promise!

## ✨ Features
- ➕ Add pizzas to cart
- ➖ Remove items
- 💰 Real-time total calculation

## 🛠️ Technologies
- ⚛️ React
- 🎨 CSS
- 📦 JavaScript

## 🚀 How to Run

1. Clone this repo
2. Run `npm install`
3. Run `npm start`
